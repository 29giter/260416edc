2022年10月22日，在中国共产党第二十次全国代表大会闭幕会上，前中国共产党中央委员会总书记胡锦涛在时任总书记习近平示意后[1]，被孔绍逊和另一名不明身份人员带离会场。当时各国新闻媒体都已进入会场，事件成为国际报道的焦点[2][3][4][5][6]，并在此后一段时间引发世界媒体各种讨论与猜测[7][8][9][10][11]。
